Game Design

Game Idea

We will be creating a top down shooter using Phaser 3 which we will be hosted on a website that we have created for it.

It will have cooperative functionality where up to two players can play at once using the same device.

The aim of the game will be to succeed against waves of enemies that will try and defeat the player.

The player will have to dodge and defeat the enemies using their weapon.

Inspirations

Some inspirations that help us shape the idea of the game include games such as Furi and Hades.

Both game feature fast gameplay, tight controls and an emphasis on dodging enemy attacks while simultaneously managing your own attacks.

They are great examples of how top down games can be easy to pick up while being very fun, interactive, and challenging with a basic concept.

There is a clear goal presented to the player and the only thing that stands between them and victory is their own skill and understanding of the game

and how the player uses the mechanics to succeed against enemies.

Controls

The controls for the game will require a keyboard with access to the arrow keys and the WASD buttons.

Up/W - Character moves up

Down/S - Character moves down

Left/A - Character moves left

Right/D - Character moves right

P/F - Character will shoot

Escape - Game will reset

Spacebar - Game will Pause/Unpause


Level Design

The levels will be from a top down perspective, so the levels will be flat with obstacles that the player will have to navigate around while avoiding enemies as they defeat them.

Some obstacles may help the player as they may be able to shoot through shorter obstacles that keep out enemies, while others will not be able to and may get in the way.

The player will be in an arena style level with limited space to move around.

Gameplay

The game will feature combat that allows the player to shoot while moving.

This means that the player will be able to shoot at enemies at a distance while moving away from them to avoid taking damage.

There will be different items that the players will have access to such as health recovery items and different upgrades.

Since it is a cooperative game, the two players will not be able to shoot each other but can still collide and block each other.
