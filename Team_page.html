Leo Tsang, Michael McAlinden & Blair Alcock

Education

We are all undergraduate students studying Computer Games Development at UWS Paisley.

We are currently in our fourth year of University and were tasked with creating a website, to host the game we have designed and developed.

We have similar previous coding knowledge in languages such as C#, Python ,HTML5 and Javascript.

We are used to using software such as Unity and Unreal Engine before, so adapting to using phaser has been fun and challenging.

Task

This website was created using Webstorm by using HTML5, CSS and Javascript.

Each person in the team is required to make a separate website but we are using the same Game Design Document which was created as a combined team effort.

The second part of the project will be creating and implementing the game into the website using Phaser.

The game will be playable on the "Game Page" webpage.

Task Breakdown

Blair was team leader, delegating tasks and offering help and solutions where needed. Blair designed much of the UI, menus and maintained version control

Leo helped create the concept of the game and played a large part in the story and characters within the game, Leo also designed the system used for the enemy collisions.

Michael was mainly involved in the level design, UI and systems creation creating the tileset, map and Menus for the game.

Each team member contributed towards the design of the game, website and documentation.

Throughout the project each member has offered assistance with issues to ensure members all have a task they can work on.

About me
